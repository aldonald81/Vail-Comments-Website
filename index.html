Vail Comments

      Designed with your tastebuds and nutritional needs in mind, Vail Comments allows users to give feedback on specific menu items at Davidson Collegeâ€™s Vail Commons in the form of upvotes and downvotes. Both users and administrators will be able to view the sentiment of specific menu items allowing dining services to make educated decisions on menu items based off the taste of students. Vail Comments' most unique feature allows users to favorite meals and receive email alerts when these meals are on the menu so you don't miss out!

      Vail Comments is the continuation of my final project from Critical Web Design with Owen Mundy. This site utilizes advanced HTML, CSS, JS, and PHP to interact with a backend MySQL database and render a user friendly front-end. The automatic mailing features are accomplished in PHP using PHPMailer and with Python using SMPT connection to Gmail. Enjoy!
